# Карточная игра в стиле Hearthstone на Python

## Автор
Груздев Николай Максимович

## Описание проекта

Данный проект представляет собой компьютерную карточную игру, разработанную на языке Python с использованием библиотеки Tkinter.

Игрок сражается против компьютерного противника, используя различные карты с уникальными эффектами.

## Возможности игры

- Выбор уровня сложности
- Система здоровья (HP)
- Система маны
- Магазин карт
- Покупка новых карт
- Компьютерный противник
- Игровой журнал событий
- Специальные эффекты карт

## Используемые технологии

- Python 3
- Tkinter
- Объектно-ориентированное программирование (ООП)

## Структура проекта

```text
main.py
cards.py
ui.py
```

## Запуск проекта

1. Установить Python 3.10 или новее.
2. Скачать проект.
3. Открыть папку проекта.
4. Запустить файл:

```bash
python main.py
```
