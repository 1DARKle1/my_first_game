# Release Notes — Card fantasi

Первый публичный релиз обновлённой версии **Dark Fantasy Card Game**.

## Что нового

- Полный **UI Upgrade**: dark fantasy дизайн, фон, иконки, стилизованные панели
- **12 уникальных иллюстраций** карт
- Переход с Tkinter на **PyQt6** — плавные анимации без подвисаний
- **Звуковые эффекты**: атака, лечение, мана, покупка, победа, поражение
- Анимации: полёт карты, hover, вспышка урона, частицы, туман на фоне
- Анимированные полоски HP и маны
- Готовый **`.exe`** — скачал и играешь, Python не нужен

## Скачать

| Файл | Описание |
|------|----------|
| `DarkFantasyCardGame.exe` | Игра для Windows (рекомендуется) |

## Как играть

1. Запустите `DarkFantasyCardGame.exe`
2. Выберите сложность: Лёгкий / Средний / Испытание
3. Кликайте по картам внизу экрана, чтобы атаковать или лечиться
4. Зарабатывайте монеты и покупайте новые карты в **МАГАЗИН**
5. Победите компьютер, обнулив его HP

## Системные требования

- Windows 10 / 11 (64-bit)
- ~200 МБ свободного места
- Разрешение экрана от 1280×720

## Известные ограничения

- Игра рассчитана на Windows; на Linux/macOS используйте запуск из исходников
- При первом запуске `.exe` создаёт служебную папку `DarkFantasyData` рядом с файлом
